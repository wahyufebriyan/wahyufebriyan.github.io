Install CodeIgniter4 di archlinux

Codeigniter merupakan framework PHP untuk membangun aplikasi berbasis web. Codeigniter sendiri memberikan 3 pilihan. yaitu: Codeigniter 2, Codeigniter 3, dan Codeigniter 4. Codeigniter 2 merupakan versi legacy dari Codeigniter, Codeigniter 3 merupakan versi terbaru dari Codeigniter, sedangkan Codeigniter 4 merupakan versi masa depan Codeigniter.

Instalasi manual


Download pada web resminya Disini..

kemudian Extract file yang sudah di download dan rename menjadi CodeIgniter4

lalu pindahkan ke directori /srv/http/ lewat teminal seperti berikut :

cd Downloads //tempat file hasil download

sudo mv CodeIgniter4 /srv/http/


instal composer :

sudo pacman -S composer

cd /srv/http/CodeIgniter4

composer install -vvv


kemudian install extension php-intl:

1. sudo pacman -Syu

2. sudo pacman -S php-intl


lalu aktifkan extension php-intl :

1. sudo nano /etc/php/php.ini

hilangkan ; pada extension=intl


selanjutnya konfigurasi vhost untuk Codeigniter :
sudo nano /etc/hosts



kemudian bikin file conf ci4.dom :
sudo nano /etc/httpd/conf/vhosts/ci4.dom



lalu aktifkan vhosts :
sudo nano /etc/httpd/conf/httpd.conf



restart httpd
sudo systemctl restart httpd

selanjutnya beri hak akses pada folder CodeIgniter4 dan writable :

cd /srv/http/

chmod 775 -R CodeIgniter4

cd CodeIgniter4

chmod 777 -R writable


Jalankan pada browser http://ci4.dom :


Semoga Bermanfaat.
